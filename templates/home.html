Coding challenge

Welcome to the ASAP away day coding challenge.

For this challenge you will work in teams to develop an interactive web application using Flask and Folium.

Below you will see a basic application - it is a Choropleth map showing the median annual pay in 2024 for each local authority in Britain.

Task 1

Add interactive labels to the map so that we can click on the map to get the data for that local authority.

Task 2

The app currently only shows data from 2024. Can you modify the app so that we can choose to view median annual pay data from 2014-2024 with an interactive element, such as a drop-down list or slider? You will also need to modify the functions inside the mapping module so that they accept additional parameters.

Extension task 1

Modify app.py so the app displays in a fullscreen window. Add CSS to style the app similar to the ONS website.

Extension task 2

Package and deploy the app to production using Waitress.
